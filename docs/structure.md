# Cangjie-CLI 代码框架结构文档

## 项目概述

Cangjie-CLI 是一个基于仓颉语言开发的AI助手命令行工具，集成了多种AI代理和工具，为用户提供智能化的命令行交互体验。

## 目录结构

```
src/
├── core/                   # 核心模块
│   ├── tool/               # 工具集合
│   │   ├── shell_tool.cj   # Shell命令执行工具
│   │   ├── fs_tool.cj      # 文件系统操作工具
│   │   └── ...             # 其他工具
│   └── code_agent.cj       # 核心代码代理
└── cli/                    # 命令行接口模块
    ├── main.cj             # 主程序入口
    └── ...                 # 其他代理
```
